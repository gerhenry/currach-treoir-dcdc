# Process Direction

## Current Direction
GF180 is being used as the open implementation track for the Treoir DC-DC program.

## Rationale
For a 3.3 V / 5 V input buck converter targeting 1.8 V output and moderate load current, standard CMOS process options are considered sufficient for first-pass implementation.

## Scope Boundary
This direction applies to low-voltage DC-DC conversion. It does not assume 20 V / 24 V high-voltage BCD-class operation.

## Parallel Track
A private 22nm FDX track remains relevant for advanced integration and differentiated product implementation.
