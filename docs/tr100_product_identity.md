# TR100 Product Identity

## Product Name
Treoir TR100

## Category
Integrated DC-DC converter / power management IC block

## First Target Use Case
- 3.3V input
- 250mA output capability
- Intended for IoT, remote sensing, and RF SoC support rails

## Positioning
TR100 is the first member of the Treoir power management family.

## Family Concept
- TR100 → 250mA
- TR200 → 500mA
- TR300 → 1A

## Internal Design Intent
TR100 is being developed as a practical, silicon-oriented DC-DC architecture including:
- UVLO
- Soft-start
- Error amplifier
- PWM control
- Driver and power stage
- External magnetics

## Status
Product identity locked.
