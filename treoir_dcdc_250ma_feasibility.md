# 250 mA Feasibility Note

The Treoir DC-DC architecture targets 1.8 V output at 250 mA from a 3.3 V / 5 V input class supply.

This current level is considered feasible in a GF180-class CMOS implementation, subject to:
- integrated switch sizing
- gate driver capability
- switching-loss analysis
- thermal validation
- external passive selection

At this stage, 250 mA is treated as a design target rather than a fully validated silicon guarantee.
